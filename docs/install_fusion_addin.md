# Install the add-in

1. Open Fusion 360.
2. Go to `Utilities > Scripts and Add-Ins`.
3. In the `Add-Ins` tab, click the `+` or `Create` option to add a new add-in folder.
4. Select `addins/CodexCADLivePreview` from this repo.
5. Run the add-in.
6. Open `Utilities > Add-Ins` and launch `Codex CAD Workbench`.

If you want the add-in to appear on startup, enable `Run on Startup` in Fusion after loading it once.
